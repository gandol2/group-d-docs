광고집행 데이터 통합 수집 시스템

네이버 광고 계정 비즈머니 조회 및 데이터 관리 도구

1. 개요

이 프로그램은 네이버 광고 API를 통해 여러 광고 계정의 비즈머니 잔액을 한 번에 조회하고 관리할 수 있는 Windows 데스크톱 애플리케이션입니다.

주요 기능

  • 광고 계정 목록 조회 및 비즈머니 잔액 확인
  • 계정별 메모 및 즐겨찾기 기능
  • 검색 및 필터링 (권한, 상태, 즐겨찾기)
  • Excel/JSON 형식 데이터 내보내기
  • 자동 업데이트

시스템 요구사항

항목 요구사항
운영체제 Windows 10/11 (64bit)
네트워크 인터넷 연결 필요
필수 정보 네이버 광고 API 자격증명 (Access License, Secret Key)

2. 설치

관리자로부터 전달받은 설치 파일을 실행합니다.

  1. 설치 파일(group-d-ad-data-collector-setup-x.x.x.exe) 실행
  2. 설치 완료 후 자동으로 프로그램 실행
참고 프로그램은 사용자 폴더에 설치되며, 관리자 권한이 필요하지 않습니다.

3. PIN 입력

프로그램 실행 시 4자리 PIN을 입력해야 합니다.

주의 PIN은 관리자에게 문의하세요. 여러 번 잘못 입력하면 프로그램이 종료됩니다.

4. 초기 설정

처음 사용 시 네이버 광고 API 자격증명을 입력해야 합니다.

입력 항목

항목 설명
Access License 네이버 광고 API 라이선스 키
Secret Key 네이버 광고 API 비밀 키
자격증명 발급 방법 네이버 광고 API 자격증명은 네이버 광고 관리 시스템에서 발급받을 수 있습니다.

5. 대시보드

로그인 후 메인 화면에서 모든 광고 계정 정보를 확인할 수 있습니다.

화면 구성

  • 상단 필터: 권한별, 상태별, 즐겨찾기 필터링
  • 검색창: 계정명, 계정번호로 검색
  • 계정 테이블: 계정 목록 및 비즈머니 잔액

비즈머니 조회

  1. 조회할 계정을 체크박스로 선택
  2. 비즈머니 조회 버튼 클릭
  3. 잔액이 테이블에 표시됨

메모 및 즐겨찾기

  • 메모: 계정별로 메모를 입력하여 관리
  • 즐겨찾기: 자주 사용하는 계정을 즐겨찾기에 추가

6. 데이터 내보내기

조회한 데이터를 Excel 또는 JSON 형식으로 내보낼 수 있습니다.

내보내기 방법

  1. 내보낼 계정을 체크박스로 선택 (또는 전체 선택)
  2. 우측 상단의 다운로드 버튼 클릭
  3. Excel 또는 JSON 형식 선택
  4. 설정된 저장 경로에 파일이 저장됨

파일 형식

형식 용도
Excel (.xlsx) 보고서 작성, 데이터 분석
JSON (.json) 다른 시스템 연동, 백업

7. 설정

우측 상단의 설정 아이콘을 클릭하여 설정 화면으로 이동합니다.

설정 항목

항목 설명
네이버 광고주센터 비밀번호 자동 로그인에 사용 (선택사항)
저장 경로 Excel/JSON 파일 저장 위치
테마 라이트/다크 모드 선택

8. 업데이트

프로그램은 자동으로 업데이트를 확인하고 설치합니다.

업데이트 과정

  1. 앱 실행 시 자동으로 업데이트 확인
  2. 새 버전이 있으면 백그라운드에서 다운로드
  3. "업데이트 준비 완료" 알림 표시
  4. 앱 종료 시 자동으로 설치
참고 업데이트 설치를 위해 앱을 완전히 종료해주세요. 다음 실행 시 새 버전이 적용됩니다.

9. FAQ

Q. API 자격증명은 어디서 발급받나요?

네이버 광고 관리 시스템(manage.searchad.naver.com)에 로그인 후, API 설정 메뉴에서 발급받을 수 있습니다.

Q. 비즈머니 조회가 안 됩니다.

다음 사항을 확인해주세요:

  • 인터넷 연결 상태
  • API 자격증명이 올바르게 입력되었는지
  • 해당 계정에 대한 조회 권한이 있는지

Q. 데이터가 저장되지 않습니다.

설정에서 저장 경로를 확인하고, 해당 폴더에 쓰기 권한이 있는지 확인해주세요.

Q. 프로그램이 느립니다.

많은 계정을 한 번에 조회하면 시간이 걸릴 수 있습니다. 필요한 계정만 선택하여 조회해주세요.
